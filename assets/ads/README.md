# Slot publicitario del Dashboard

  Para activar el banner publicitario:

  1. Copiá tu imagen aquí (PNG / JPG / WebP recomendado).
  2. Editá `lib/ads.ts` y cambiá `enabled: true`.
  3. Apuntá `image` con `require("../assets/ads/tu-banner.png")`.
  4. Configurá `link.type` (`"url"` o `"whatsapp"`) y completá los datos.

  Tamaño sugerido: 1080 x 540 px (relación 2:1).
